# 🌏 tabisuke（タビスケ）

## 公式サイトはこちら https://tabisuke.click/

**旅をもっとスマートに、もっと楽しく。**  
**tabisuke** は、「たびのしおりを簡単に」をコンセプトに

旅行やお出かけの計画・管理をサポートする旅行管理アプリです。  
行きたい場所やスケジュールを手軽に整理し、思い出をより豊かに残すことができます。


---

## 🚀 プロジェクト概要

- **アプリ名**：tabisuke（タビスケ）  
- **開発期間**：1〜2ヶ月  
- **開発環境**：Kotlin（Android）  
- **バックエンド**：Firebase（Authentication / Firestore / Storage）  
- **特徴**：AI駆動開発  

---

## ✨ 主な機能

- 🗺️ **旅のスケジュール管理**  
  行き先・日付・メモを簡単に追加し、「しおり」で確認可能。

- 👥 **簡単に共有1**  
  URLやQRコードをスキャンするだけで、簡単に友達と「しおり」を共有可能。

- 👥 **簡単に共有2**  
  アプリを入れていなくても、Web版としてブラウザでも見ることが可能。  

- 🔥 **Firebase連携**  
  ユーザー認証やデータ保存をFirebaseで安全に管理。

- 📸 **思い出の共有**  
  旅行での行き先の写真やメモを保存し、後で振り返ることが可能。
